# EDA-FIFA

## Objective:
- To perform Exploratory Data Analysis on Player Stats from the famouse football video game FIFA 23 and FIFA 22.
- Gain insights on player body types, height, weight, progress in overall and potential stats over the year, etc.
- To compile a report in Power Bi that summarises results and insights gathered from EDA.


Dataset:
<a href="https://www.kaggle.com/datasets/bryanb/fifa-player-stats-database">FIFA23 OFFICIAL DATASET</a> by <a href="https://www.kaggle.com/bryanb">BryanB</a>
